# 🧠 Neurix Frontend

A modern AI SaaS frontend built with React, TypeScript, Vite, and Tailwind CSS.

## ✨ Features

- Modern Landing Page
- Authentication UI
- Dashboard
- Chat Interface
- Document Management
- Upload UI
- Responsive Design
- Dark Mode (Coming Soon)

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- React Icons
- clsx
- tailwind-merge

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/codeid9/Neurix.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build

```bash
npm run build
```

Preview

```bash
npm run preview
```
